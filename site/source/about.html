关于 Audi Joiner 小组件

Audi Joiner 是开源免费的，由 Audi 车主兴趣开发，所有责任与一汽奥迪公司无关。

Audi Joiner 不会收集您的个人账户信息，所有账号信息将存在 iCloud 或者 iPhone 上但也请您妥善保管自己的账号。

Audi Joiner 会不定期推出新功能，如果一汽奥迪官方推出了小组件，Audi Joiner 将会停止更新与支持。同样收到一汽奥迪警告也会停止更新支持。

代码开源

关于有志之士加入开源社区，添加自己一份贡献，同样有更好的意见或者问题反馈在 GitHub 提出！
